💾 Exportar datos
Guarda en tu carpeta Documents → iCloud

Exporta todos tus datos en formato JSON. Guárdalos en Documents/MyAlquimia/ para que iCloud los respalde automáticamente.

📂 Importar respaldo
Restaura desde un archivo JSON exportado

Selecciona un archivo .json exportado previamente para restaurar todos tus datos. Esto reemplazará los datos actuales.

📋 Instrucciones de respaldo con iCloud
  1. Haz clic en Exportar JSON completo
  2. Cuando el navegador descargue el archivo, muévelo a Documentos → MyAlquimia
  3. iCloud sincroniza esa carpeta automáticamente a la nube ☁️
  4. Para restaurar: usa Importar respaldo y selecciona el .json
  5. Recomendamos hacer esto al menos una vez por semana
